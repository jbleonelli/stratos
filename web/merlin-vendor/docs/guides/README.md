## When to add here

How-to or playbook content — user-facing or operator-facing. If the audience is "someone using Merlin to do their job", it belongs here.

Engineering reference (enums, fixtures) goes in [`../reference/`](../reference/). System-design docs go in [`../architecture/`](../architecture/).
