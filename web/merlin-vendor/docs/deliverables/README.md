## Regenerate

```bash
# From repo root (first time: creates .venv-deliverables)
python3 -m venv .venv-deliverables
.venv-deliverables/bin/pip install -r scripts/requirements-deliverables.txt
.venv-deliverables/bin/python scripts/generate-merlin-architecture-doc.py

# PDF also requires pandoc + typst (brew install pandoc typst)
pandoc docs/deliverables/Merlin-Technical-Appendix.md \
  -o docs/deliverables/Merlin-Technical-Appendix.pdf --pdf-engine=typst
```
